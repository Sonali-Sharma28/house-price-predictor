# 🏠 House Price Predictor

My first machine learning project.
Predicted California house prices using 3 ML models.

## What I built
- Explored 20,640 real houses using Pandas & Matplotlib  
- Discovered income & location drive prices the most
- Built and compared 3 models:
  - Linear Regression  
  - Random Forest  
  - Gradient Boosting

## Skills used
Python · Pandas · NumPy · Matplotlib · Scikit-learn · Jupyter

## This project covers
✅ Data exploration & visualization
✅ Correlation analysis
✅ Feature scaling
✅ Train/test split
✅ 3 ML models compared
✅ Feature importance
✅ Price prediction function
